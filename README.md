# 👥 Azure Active Directory (AAD) Identity Lab

This lab demonstrates how to manage cloud identity and access using **Azure Active Directory**, including user and group creation, Multi-Factor Authentication (MFA), and role assignment — all within the free tier of Microsoft Entra ID (Azure AD).

---

## 🧰 Technologies Used

- Microsoft Azure Portal
- Azure Active Directory (Microsoft Entra ID)
- Directory Roles
- Multi-Factor Authentication (MFA)
- Groups & Users Management

---

## ✅ What Was Done

- Created two Azure AD users: `john.smith` and `emma.jones`
- Created a security group: `CloudAdmins`
- Added both users to the group
- Enabled MFA for `john.smith` via AAD portal
- Assigned the `User Administrator` directory role to `emma.jones`
